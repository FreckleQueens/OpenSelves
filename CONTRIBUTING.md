# Contributing guidelines

1. Before undertaking any big task, please reach out to us& via discord or email (see README) to organize work.
2. We& take a firm stand **against** the use of generative AI in any part of the process whatsoever.
3. When submitting a pull request, your code will be integrated to the project and will be published under the project's
[LICENSE](LICENSE).
4. If you wish, you can add (or remove) your name in the [CONTRIBUTORS](CONTRIBUTORS) file. This must be done in its own
separate commit.
